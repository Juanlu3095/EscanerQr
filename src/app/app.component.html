Instrucciones de uso

Este escáner permite al usuario registrar los datos guardados en un QR. Presione 'Start' para empezar. Una vez el escáner reconozca el código QR, se mostrarán en pantalla los datos del registro. En ese momento no se podrán registar más hasta que pulse en 'Reset'. Una vez presionado dicho botón, podrá volver a escanear otro código QR. Para detener el escáner, presione 'Stop'.
